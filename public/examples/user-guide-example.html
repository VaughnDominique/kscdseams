Main Application Content

This is where your main application content would be. The user guide will appear in a sidebar when the help button is clicked.

User Guide Toggle Button Options

Default (bottom right):

const userGuide = new UserGuide();

With text label:

const userGuide = new UserGuide({
    toggleButton: {
        showLabel: true,
        labelText: 'Help Guide'
    }
});

Different position:

const userGuide = new UserGuide({
    toggleButton: {
        position: 'top-right'  // 'top-right', 'top-left', 'bottom-left', 'bottom-right'
    }
});

In navigation bar:

const userGuide = new UserGuide({
    toggleButton: {
        position: 'fixed-nav',
        showLabel: true,
        icon: 'bi-info-circle'
    }
});
document.getElementById('nav-guide-placeholder').appendChild(userGuide.getToggleButton());

In sidenav (above logout button):

const userGuide = new UserGuide({
    toggleButton: {
        position: 'fixed-nav',
        showLabel: true,
        icon: 'bi-question-circle',
        labelText: 'Help'
    }
});
// Add the guide button above logout in sidenav
const logoutButton = document.querySelector('.sidenav-logout');
const sidenavMenu = document.querySelector('.sidenav-menu');
sidenavMenu.insertBefore(userGuide.getToggleButton(), logoutButton);
